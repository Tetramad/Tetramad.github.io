Day2: chrome extension으로 사용자가 보고있는 페이지 수정하기

2020-08-06

사용자가 보고 있는 페이지 접근하기

사용자가 보고 있는 페이지에 접근하려면 chrome.tabs.execute... API를 이용하거나 manifest.json 파일의 "content_scripts" 부분을 이용해야한다.

사용자가 보고 있는 페이지 접근하기

  • chrome.tabs.executeFile 또는 chrome.tabs.executeScript를 이용해서 실행하기
  • "content_scripts"에 실행할 파일 추가하기

위 두 가지 중 하나의 방법으로 실행한 코드만 사용자가 보고 있는 페이지에 접근할 수 있다.

수정해보기

접근하는 것을 성공하면 수정하는 것은 JS로 HTML 수정하는 것이 전부이다.
