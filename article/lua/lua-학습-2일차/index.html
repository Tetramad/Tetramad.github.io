function

function과 variable이 같은 namespace를 사용한다. shadowing도 가능하니 이 점을 주의해야한다.

function max(a, b)
    return (a > b) and a or b
end
max = 1
print(max) -- 1

anonymouse function도 지원한다.

local max = function (a, b)
    return (a > b) and a or b
end

하지만, 문법이 인덱스가 1부터 시작하는 것을 잘 이용하고 있고, 잘 지원해 주고 있어 그리 나쁜 느낌은 아니다.

named parameter

파라미터로 table을 넘겨주는 특별한 방법을 허용한다.

function rename(args)
    os.rename(args.old, args.new)
end

rename{ old = "temp.lua", new = "perm.lua" }
