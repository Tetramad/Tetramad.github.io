C float increment/decrement operator

2020-04-17

float형과 double형에도 사용할 수 있다. ++연산자는 += 1 같은 효과를 내고, --연산자는 -= 1 같은 효과를 낸다. 'postfix'와 'prefix'의 성질은 유지한다.

float f = 0.0f;
++f; /* 1.0f */
--f; /* 0.0f */
