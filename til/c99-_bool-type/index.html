C99 _Bool Type

2020-04-18

_Bool형은 부호가 없는 정수형(unsigned integer type)이다. 랭크(rank)가 가장 낮다. 0과 1만을 저장할 수 있다. 0이 아닌 값을 저장할 경우 1을 저장한다. 따라서, 일반적인 형변환(type conversion)과 다르게 동작할 수 있다.

int should_zero = (int)0.5;
_Bool should_one = (_Bool)0.5;
