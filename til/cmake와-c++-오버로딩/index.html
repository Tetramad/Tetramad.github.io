CMake와 C++ 오버로딩

2019-12-13

CMake 모듈 중 CheckCXXSymbolExists에서 check_cxx_symbol_exists메크로는 아래와 같은 코드를 생성하여 확인한다. 그렇기에 C++의 오버로딩된 함수등을 판별할 수 없다.

// check_cxx_symbol_exists(<symbol> <files> <variable>)
/* */
#include <file>
#include <file>
...
#include <file>

int main(int argc, char** argv)
{
  (void)argv;
#ifndef symbol
  return ((int*)(&symbol))[argc];
#else
  (void)argc;
  return 0;
#endif
}
// return 0 -> not found, return not 0 -> found.
