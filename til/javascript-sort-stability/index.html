Javascript Sort Stability

2020-05-07

Stability

Stability란 정렬 알고리즘에서 정렬 이후 같은 값 또는 팩터(factor)를 갖는 요소끼리는 정렬 이전과 같은 순서를 유지하는 가를 나타내는 말이다. 유지되면 stable sort라 부르고, 유지되지 않으면 unstable sort라 한다. 대표적인 stable sort는 merge sort, bubble sort, insertion sort 등이 있고, 대표적인 unstable sort는 quick sort, heap sort, selection sort가 있다.

Javascript Array#sort

표준을 살펴보면 정렬의 stability는 구현하는 사람이 선택할 수 있도록 하였다가 최근에 stable하도록 정의되었다. 구체적으로는 ECMAscript 2019(ES10)부터 정렬은 stable하다. 예외적으로 Chrome 브라우저는 더 이전부터 정렬을 stable sort로 구현했다고 한다. stability가 중요한 상황에서는 주의하도록 하자.
