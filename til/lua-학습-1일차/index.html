논리 값 true, false와 논리 연산자 and, or not

false와 nil을 제외한 모든 값은 true로 취급한다. 따라서 C랑 다르게, JavaScript랑 다르게, 0과 빈 문자열("")도 true 값을 나타낼 수 있다.

and 연산자는 왼쪽 값이 true를 나타내면 오른쪽 값이 된다. 왼쪽 값이 false를 나타내면 왼쪽 값이 된다.

or 연산자는 왼쪽 값이 false를 나타내면 오른쪽 값이 되고, true를 나타내면 왼쪽 값이 된다.

not 연산자의 결과는 무조건 boolean 값이다.

and와 or의 처리 방식을 이용해서 C의 ternary operator(?:)같은 코드를 만들 수 있다.

function max(a, b)
    return (a > b) and a or b
end

number

기본 자료형 중 수를 나타내는 자료형으로 number가 존재한다. double-precision floating point number라고 한다. 즉, 64비트 float형이다. 이 자료형을 기본적으로 사용하는 이유 중 하나는 64비트 float형은 32비트 정수형의 모든 값을 오차 없이 나타낼 수 있기 때문이라고 한다.
